## Coverage Generation

Coverage data is generated by the ingestion pipeline (`scripts/ingest/build-dataset.mjs`) and stored in:

- `data/coverage-report.json` — machine-readable coverage summary
- `data/coverage.json` — alias for fleet tooling compatibility
- `data/domain-dataset.json` — full dataset including `_meta.coverage`

To regenerate: `npm run ingest`
